HUL Financial Analysis Dashboard - Power BI

Overview

This project is a financial analysis dashboard developed in Power BI using Hindustan Unilever Limited (HUL) financial data. The dashboard transforms financial statements into interactive visualizations, enabling users to analyze company performance, financial health, profitability, liquidity, and cash flow trends.

The objective of this project is to demonstrate how financial modeling and business intelligence can be combined to support data-driven decision-making.

Features

* Interactive financial dashboard
* Balance Sheet analysis
* Income Statement analysis
* Cash Flow Statement analysis
* Key Performance Indicators (KPIs)
* Financial ratio analysis
* Year-over-year trend analysis
* Dynamic filtering and drill-down capabilities
* Executive-level business insights
